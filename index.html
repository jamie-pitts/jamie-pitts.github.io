Welcome!

I am a Senior Software Engineer working at the British Broadcasting Corporation, based in London. I work in an agile-orientated team that provides the back-end systems for the News department of the corporation. We create products such as the various election results system's, and the API behind the BBC News mobile applications. These products are mainly cloud-based Java servlets, which use the RESTful architectural style - making use of technologies such as; Amazon Web Services, Spring, and PostsreSQL. I have acted as technical lead for a new project, and have helped out in designing the technical architecture for our projects.

A previous Graduate Software Developer at the BBC, which involved working in various different teams and areas around the corporation, allowing me to gain knowledge of various different software systems. Projects worked on include; the cloud-based Java back end system that processes all the data used by online and red button for the BBC's General Election 2015 coverage, working as an Android developer, creating and integrating Android Wear functionality into the main BBC News application, and working on a Scala cloud-based product that allows for BBC staff to log into tools based websites whilst off the BBC network.

Previous attendee of the University of Birmingham; studying Artificial Intelligence and Computer Science, and graduating with a First Class (Honours) degree. Some of my final year modules included; Evolutionary Computation, Databases 2, Intelligent Robotics, Operating Systems with C/C++, Networks and Distributed Systems, and a year long Software Project. I am now currently working part-time towards an MSc in Software Engineering and Internet Architecture.
